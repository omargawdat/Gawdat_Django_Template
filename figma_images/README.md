## How to Download Individual Assets

### Method 1: Using Figma Desktop (Recommended)
1. Open the Figma file in Figma Desktop app
2. Select the specific frame/icon/image you want to export
3. Note the node ID from the URL or properties panel
4. I can then download it using the node ID

### Method 2: Using Figma Export Feature
1. Right-click on any asset in Figma
2. Select "Export" or "Copy as PNG/SVG"
3. Save directly to this folder

### Method 3: Using Figma API (Automated)
Use the provided `download_figma_assets.py` script:
```bash
export FIGMA_TOKEN="your-token-here"
python3 download_figma_assets.py
```

Get your Figma token from: https://www.figma.com/settings
